Your file system is a mess of identical yellow folders. Finding the right directory means reading dozens of folder names, scanning through endless lists, and wasting valuable time every single day.

## The Solution: FeColorizer

FeColorizer automatically color-codes your folders based on their names, creating an instantly recognizable visual hierarchy in File Explorer. No more scanning text—just glance and navigate.

### One Right-Click. Instant Organization.

Right-click any folder. Select "Colorize subfolders." Done. Every subfolder gets a unique color based on its first letter, transforming chaos into visual clarity in seconds.

## Frequently Asked Questions

### Is it safe?
Yes. FeColorizer only modifies folder metadata (icons). It never touches your actual files, never connects to the internet, and never collects data.

### Will it slow down my computer?
No. FeColorizer has zero performance impact. It only runs when you explicitly invoke it via right-click.

### Does it require administrator rights?
Only during installation to register the context menu. After that, normal use requires no special permissions.

### Can I use it on network drives?
Yes, but icon paths are absolute, so colored folders only display correctly on the PC where FeColorizer is installed.

### What if I reinstall Windows?
Simply reinstall FeColorizer and run it on folders you want to re-color. Your folder structure remains unchanged.

### Is my data private?
Completely. FeColorizer is offline-only. No telemetry, no analytics, no phone-home features.
